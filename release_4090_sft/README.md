# Structure-Aware LLM

This project compares a baseline Transformer with a structure-aware Transformer on JSON and Markdown generation tasks.

## Directory

- `configs/`: experiment configuration files
- `data_gen/`: synthetic data generation and dataset construction
- `tokenizer/`: tokenization utilities
- `parser/`: structural state extraction and annotation
- `model/`: model definitions
- `train/`: training scripts and dataset loaders
- `eval/`: evaluation scripts
- `results/`: tables and qualitative cases
